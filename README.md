🚗 Washly | On-Demand Car Wash Platform

Washly is a production-grade, full-stack web application designed to modernize the car wash and detailing industry. Inspired by platforms like Uber, it delivers a seamless booking experience for customers while providing a powerful administrative system for business control and scalability.

Built with a focus on real-world business logic, performance, and user experience, Washly demonstrates advanced full-stack development capabilities.

🚀 Core Features
🔹 Dual-Interface System

A complete ecosystem with:

Client Portal → Easy booking, service selection, and request tracking
Admin Dashboard → Full control over users, services, and requests
🔹 Real-Time Request Management

A custom-built workflow that allows administrators to:

Instantly review incoming bookings
Accept or reject requests in real time
Manage service flow efficiently
🔹 Dynamic Service Management
Add, edit, or remove services بسهولة
Control pricing and availability
Support for Basic vs Premium packages

🔹 Secure System Architecture
